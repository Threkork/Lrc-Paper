# Lrc-Paper

一个lrc歌词打轴编辑器。
目标是简洁且好用的lrc歌词打轴编辑。

初期版本，所以非常简陋。

#### 实现 / 尚未实现：

- [x] 导入lrc
- [x] 音频播放控制
- [ ] 本地缓存
- [ ] 快捷键操作
- [ ] 多语言
- [ ] 翻译模式

#### You can

```shell
git clone https://github.com/Threkork/Lrc-Paper.git
```

```shell
npm install
```
then
```shell
npm run dev
```
or
```shell
npm run build
```
